The Mission

Making AI coding accessible, open, and alive

Every developer deserves a powerful AI coding partner — not locked behind expensive subscriptions or closed ecosystems. Lorapok brings the full power of modern AI agents to your JetBrains IDE, for free.

The Problem

AI IDEs are holding developers back

Closed Ecosystems

Most AI coding tools lock you into a single provider with no escape hatch. Your workflow depends on one company's decisions.

Expensive Access

Premium AI features cost $20-50/month per developer. Teams pay thousands for capabilities that should be accessible to everyone.

Limited Extensibility

No MCP support, no custom tools, no workflow automation. You get what they ship — nothing more.

The Solution

That's why we built Intellij Lorapok

An open-source AI agent IDE that brings multi-provider intelligence, sub-agent orchestration, and extensible powers to every JetBrains IDE — at zero cost.

Features

Everything an AI IDE should have

🤖

Sub-Agent Orchestration

Decompose complex tasks into parallel sub-agents. Up to 5 concurrent agents working on your code simultaneously.

🔌

MCP Protocol

Full Model Context Protocol support. Expose your IDE as an MCP server and connect to external MCP tools.

⚡

Workflow Engine

n8n-style visual automation. Chain triggers, actions, conditions, and AI nodes into reusable pipelines.

🧠

44 AI Powers

Code generation, refactoring, debugging, testing, documentation, git, web search, media generation, and more.

☁️

Cloud Knowledge

Deep AWS and Azure expertise built in. Architecture design, IaC generation, cost optimization, security review.

🤝

AI Collaboration

Connect multiple AI agents. Round-table discussions, second opinions, and parallel problem-solving across AIs.

How It Works

Up and running in 3 steps

01

Install the Plugin

Get Intellij Lorapok from the JetBrains Marketplace. Works with all JetBrains IDEs.

02

Configure Providers

Add your API keys — or use free providers. Supports OpenAI, Anthropic, Gemini, Groq, Mistral, Ollama, and more.

03

Start Coding with AI

Open the Lorapok panel and start asking. The agent handles the rest — sub-agents, tools, context, and code.
