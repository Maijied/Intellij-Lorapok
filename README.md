## What is Intellij Lorapok?

A Lorapok Labs product — an AI coding agent for JetBrains IDEs with:

- 🤖 Sub-agent orchestration (up to 5 concurrent agents)
- 🔌 Full MCP Protocol support (server + client)
- ⚡ n8n-style workflow automation engine
- 🧠 44 AI-powered coding capabilities
- ☁️ Deep AWS & Azure knowledge
- 🤝 AI-to-AI collaboration hub
- 🆓 Free LLM provider integration (zero-cost operation)
- 🎨 Multi-modal content generation (text, image, audio, video)
- 📦 OpenClaw integration for background workers
- 🏗️ Senior architect persona with full-stack expertise

## Supported Providers

OpenAI, Anthropic Claude, Google Gemini, OpenRouter, Groq, Mistral, DeepSeek, Ollama (local), and Cursor provider.
